Features

💬

Posts & Timelines

Create, read, search, and engage with posts. View home, user, and mention timelines.

👥

Users & Social

Follow, block, mute. View followers and following. Manage your social graph.

🤖

Grok AI

Summarize threads, analyze content, draft posts, and get suggested replies.

📩

Lists & DMs

Create and manage lists. Send and receive direct messages.

🎨

Media Upload

Upload images and videos with alt text. Automatic chunked upload for large files.

🎧

Spaces

Search and discover live audio Spaces. View space details and participants.

Quick Start

# Authenticate with your X account
x auth login

# Check your auth status
x me

# Post something
x post create "Hello from x-cli!"

# View your timeline
x timeline home

# Search posts
x search "bun runtime" --limit 10

# Ask Grok to summarize a user
x grok summarize @bunjs

# Interactive mode
x -i

Command Reference

Authentication

  • x auth login
  • x auth logout
  • x auth status
  • x me

Posts

  • x post create <text>
  • x post get <id>
  • x post delete <id>
  • x post reply <id> <text>
  • x post quote <id> <text>

Timelines

  • x timeline home

  • x timeline mentions
  • x search <query>

Engagement

  • x like <id>

  • x bookmark list

Users

  • x user <username>
  • x follow <username>
  • x followers
  • x following
  • x block <username>

Grok AI

  • x grok "<query>"
  • x grok summarize <id>
  • x grok analyze <id>
  • x grok draft <topic>
